ショートカットキー & 操作ヘルプ

マインドマップ操作

  • Enter : 同一階層（兄弟）にノードを追加
  • Tab : 下位階層（子）にノードを追加
  • Delete / Backspace : 選択したノードを削除
  • F2 : テキストを編集
  • Ctrl + Z : 元に戻す (Undo)
  • Ctrl + Y : やり直し (Redo)

キャンバス操作

  • ドラッグ (空白地) : キャンバスの移動 (パン)
  • ホイール回転 : 拡大・縮小 (ズーム)
  • ダブルクリック (空白地) : 新規ルートノードの追加
  • ドラッグ (ノード上) : ノードの移動
  • 接続線の中間の「＋」をクリック : ノードの間に新規ノードを挿入
