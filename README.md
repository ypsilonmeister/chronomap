# ChronoMap

ChronoMap は、思考のプロセスを時間軸と共に記録・再生できる、ローカルファーストなマインドマップWebアプリケーションです。

子どもの日々の発見やひらめきを、音声と写真で直感的にキャンバスに配置し、後からその「思考が広がっていく過程」をタイムラインとして追体験することを目的に作成しました。

---

## 主な機能

- **タイムライン・プレイバック**: ノードが追加されていく過程を、時間を圧縮してアニメーション再生します。
- **直感的なメディア入力**: ノードをタップして、キーボードを介さずに音声入力（文字起こし）や写真の添付が可能です。
- **ローカルファースト (IndexedDB)**: 画像や音声を含むすべてのデータはブラウザ内に保存され、完全なオフライン環境でもサクサク動作します。
- **重なりを回避する自動レイアウト**: 思考を妨げないよう、ノードが増えても自動で美しい放射状に配置されます。

---

## ローカルでの動かし方

Node.js (v20以上) がインストールされた環境で、以下のコマンドを実行してください。

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```
